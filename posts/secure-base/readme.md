## 概览

在业界的安全实践中，有非常复杂的安全基础设施，以及相关的标准，导致衍生出来了非常多个概览并且难以理解，例如：

- X.509 Certificate
- DER
- PEM
- ASN.1
- PKCS
- ...

本文试图学习这些安全基础设施，并进行记录，方便后续翻阅。

## 概览

RUP 4+1 表示了 5 种视图，如下所示：

![rup41](assets/rup41.png)

视图描述如下表所示：

视图名称 | 中文名 | 视图描述
-|-|-
Logic View | 逻辑视图 | -
Process View | 进程视图 | -
Development View | 开发视图 | -
Physical View | 物理视图 | -
Scenarios | 用例视图 | -

## 5+1 视图
